A Joint Minimization Framework for Transformer Interpretability Information Crystallization

Saqib Nazir Bhat · March 2026 · Independent research · CC BY 4.0
GitHub LaTeX source Experiments Cite
Abstract. Transformer interpretability is usually posed as: identify the parameters and inputs responsible for a model's behaviour. Existing methods work top-down, searching the combinatorial space by ablation from the full model. I propose a different formulation. Given an observed output token $\hat{x}$, find the minimal $S \subseteq [d]$ and $T \subseteq [n]$ such that the argmax over the restricted model still equals $\hat{x}$. A single observation simplifies the problem: softmax does not affect argmax. Preserving the predicted token requires only that the logit margin $\delta$ remains positive, which is strictly weaker than matching the full distribution. Under this relaxation I introduce Information Crystallization, a bottom-up $O(k \cdot d)$ algorithm (seed, greedy growth, prune) that finds local minima of $(S, T)$ jointly. I decompose $S$ into a final-block component $S_L$ and a residual-stream support $S_{\text{supp}}$, connect the sparsity conjecture to the Lottery Ticket Hypothesis, and identify the $T$-dependence of $S^*$ as the central open problem. I have run the seed phase of the algorithm on GPT-2 small (N = 17 stratified sequences) as a feasibility check. Only the seed step is tested; growth and prune phases are future work.

1. Problem

Let $V$ be a finite vocabulary and $X = (x_1, \ldots, x_n) \in V^n$ an input sequence. A language model parameterized by $\theta \in \mathbb{R}^d$ defines $$ P_\theta(x_{n+1} \mid x_1, \ldots, x_n) = \mathrm{softmax}\!\big(f_\theta(X)\big), $$ where $f_\theta : V^n \to \mathbb{R}^{|V|}$ composes $L$ non-linear transformations $f_\theta = f_\theta^{(L)} \circ \cdots \circ f_\theta^{(1)}$. The predicted token is $\hat{x} = \arg\max_{v \in V} P_\theta(v \mid X)$.

Problem. Given $\hat{x}$, find minimal $S \subseteq [d]$ and $T \subseteq [n]$ with $$ \arg\max_{v \in V}\; P_{\theta_S}(v \mid X_T) \;=\; \hat{x}. $$ $\theta_S$ denotes $\theta$ restricted to indices $S$; $X_T$ denotes $X$ restricted to positions $T$. For $d \sim 10^{11}$ the problem is intractable in general. The next section weakens the preservation condition and makes it tractable under a sparsity assumption.

2. The Softmax Relaxation

Define the logit margin $$ \delta \;=\; f_\theta(X)[\hat{x}] \;-\; \max_{v \neq \hat{x}} f_\theta(X)[v]. $$ The argmax condition is equivalent to $\delta > 0$. Softmax is strictly monotone, so preserving the argmax requires only preserving the sign of $\delta$. No particular probability value needs to match. This is the key relaxation: a distribution-matching condition becomes a margin inequality.

Most parameters do not determine the identity of the winning token: they refine losing-token probabilities, smooth the distribution, or encode features for positions outside $T$. A parameter $\theta_i$ is irrelevant to $\hat{x}$ if perturbing it changes $\delta$ by less than $\delta$ itself. The set of irrelevant parameters is large because $\delta > 0$ creates a $\delta$-wide buffer around the decision boundary.

Sufficient Condition. For all $\theta_i \notin S$, $\bigl| \mathrm{logit}_{\hat{x}}(\theta_S, X_T) - \mathrm{logit}_{\hat{x}}(\theta, X) \bigr| < \delta$. A proof via Lipschitz continuity of $f_\theta$ or a first-order perturbation bound would suffice.

3. Information Crystallization

Top-down methods search $O(2^d)$ by ablating from $|S| = d$. I invert the direction of search: start with $|S| = 1$ and grow. If the true minimal set is small, growing finds it in time proportional to its size, not the model's.

Algorithm: Information Crystallization
  1. $g \gets \nabla_\theta \log P_\theta(\hat{x} \mid X)$
  2. $S \gets \{\arg\max_j |g_j|\},\quad T \gets \{\arg\max_t \|\partial f_\theta / \partial x_t\|\}$
  3. while $\arg\max_v P_{\theta_S}(v \mid X_T) \neq \hat{x}$:
    • $\hat{g} \gets \nabla_\theta \log P_{\theta_S}(\hat{x} \mid X_T)$
    • $S \gets S \cup \{\arg\max_{j \notin S} |\hat{g}_j|\}$
  4. for each $s \in S$: if argmax is preserved without $s$, drop $s$
  5. if argmax broken: goto 3
  6. return $(S, T)$

Complexity. Seed: $O(d)$. Each growth step: $O(|S|)$ for the restricted pass plus $O(d)$ to rank candidates via the gradient. After $k$ growth steps: $O(k \cdot d)$. Prune: $O(k)$ forward passes. Total $O(k \cdot d)$, tractable when $k \ll d$.

Three properties distinguish the approach.

  • Direction of search. Existing methods start from the full model and remove components. This algorithm starts from empty and adds.
  • Gradient-guided growth. Ranking $d$ candidates per step would cost $O(d^2)$ total; using the restricted-subnetwork gradient as a ranking heuristic reduces each step to $O(d)$.
  • Joint optimization. $S$ and $T$ grow simultaneously, capturing interactions between which parameters and which inputs matter.

4. Minimizing $T$

Let $\alpha^{h,\ell}(t)$ denote the attention weight on position $t$ from head $h$ at layer $\ell$. Define cumulative salience $\sigma(t) = \sum_{h,\ell} \alpha^{h,\ell}(t)$. For threshold $\varepsilon$ chosen so that positions with $\sigma(t) < \varepsilon$ contribute less than $\delta/2$ to the winning logit, $$ T_{\text{sal}} = \{\, t \in [n] : \sigma(t) > \varepsilon \,\}. $$ I conjecture $T = \{x_{n-k}, \ldots, x_n\} \cup T_{\text{sal}}$ with $k \ll n$.

Failure modes of the suffix assumption. Transformers use global attention; a position-1 token with high $\sigma$ can dominate the logit landscape. Three concrete cases:

  • Sentence-initial negation. “Despite everything, the answer is still…” Position 1 reshapes the entire distribution.
  • Long-range syntactic dependencies. The subject at positions 1–3 may determine verb form 20+ tokens later.
  • Named entities. In “Harry Potter picked up his…” the entity at positions 1–2 causes “wand” over “phone”.

The corrected claim: $T$ is identified by attention weight concentration, not positional recency.

5. Minimizing $S$

I decompose $S = S_L \cup S_{\text{supp}}$: $S_L$ lives in the final transformer block and directly governs $\delta$ (unembedding projections for $\hat{x}$ and its nearest competitor, top-activating attention heads in the last layers, a small MLP set); $S_{\text{supp}}$ consists of earlier-layer parameters whose representations flow into $S_L$ via the residual stream [Elhage et al. 2021]. The contribution of $S_{\text{supp}}$ must satisfy $|\delta(S_L) - \delta(S_L \cup S_{\text{supp}})| < \delta/2$.

Two external results support this decomposition. Frankle & Carlin's Lottery Ticket Hypothesis establishes that dense networks contain sparse trainable subnetworks; the conjecture here is the inference-time analogue. Meng et al.'s ROME locates factual associations in middle-layer MLP modules, which means $S_{\text{supp}}$ can be substantially larger for factual recall than for syntactic continuation. Any empirical protocol must stratify by query type.

6. Coupling

$S$ and $T$ are not independently minimizable. When $T$ changes, the embeddings fed to layer 1 change; the perturbation propagates through all $L$ layers via the residual stream; the set of load-bearing parameters in $S_L$ shifts accordingly. Formally, $$ S^*(T) \;=\; \arg\min_S |S| \quad \text{s.t.} \quad \arg\max_v P_{\theta_S}(v \mid X_T) = \hat{x}. $$ Treating $S$ and $T$ as independent is an approximation; its error is small only when $S_{\text{supp}}$ is small relative to $\delta$.

Open Problem. Characterize $T \mapsto S^*(T)$. (i) Bound $|S^*(T) - S^*(\theta, X)|$ as a function of $|T|$. (ii) Determine whether greedy decomposition is within a constant factor of the true joint minimum.

7. Empirical Protocol (GPT-2)

GPT-2 is suitable for initial verification because its full parameters and attention weights are publicly accessible. The protocol below has not yet been executed.

  1. Sampling. 1,000 sequences stratified by query type: syntactic continuation, factual recall, negation-heavy, long-range syntactic dependency.
  2. Baseline. Record $\hat{x}$ for each sequence under the full model.
  3. $T$ identification. Greedily remove positions in ascending order of $\sigma(t)$; record the minimal $T$ at which $\hat{x}$ flips.
  4. $S$ identification. Greedily ablate parameters in ascending order of gradient sensitivity; record the minimal $S$ at which $\hat{x}$ flips.
  5. Size measurement. Report $|S|/d$ and $|T|/n$ per query type (mean, median, 90th percentile).
  6. Coupling test. Recompute $S^*(T)$ for each $T$ from Step 3 and compare to $S^*(\theta, X)$.

Preliminary seed-phase results. I have run Step 1 of the algorithm on N = 17 hand-curated GPT-2 small sequences, stratified into four query types. The input-position seed $t^*$ landed on the final token in only 11.8% of sequences. On long-range sequences it clustered near the subject noun (mean distance from the end, 7.67 tokens over three examples). This is evidence against a suffix-only heuristic. The parameter seed $j^*$ was the tied wte.weight tensor in 17/17 sequences, which is consistent with the $S_L$ conjecture but is not a clean test: on GPT-2 the input embedding and unembedding projection share one parameter tensor. A model with untied embeddings (for example Pythia-160M) is the obvious next experiment. Full notebook and stratified results are in experiments/seed_phase.ipynb and experiments/results.md.

8. Limitations

  • The suffix approximation is not a theorem.
  • $|S_{\text{supp}}|$ is not formally bounded.
  • Greedy decomposition is suboptimal in general; the gap to the true joint minimum is uncharacterized.
  • The Sufficient Condition is unproved.
  • GPT-2 results do not automatically generalize to frontier-scale models.
  • Only the seed phase has been run. Growth, prune, and verification phases of Algorithm 1 are not implemented. N = 17 is a sanity check, not a study.
  • Weight-tying on GPT-2 blocks a clean $S_L$ test. wte.weight serves as both input embedding and unembedding projection, so the $j^*$ result does not adjudicate the decomposition.
